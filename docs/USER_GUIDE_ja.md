# ユーザーガイド

## このシステムについて

研究室のGPUサーバーや部屋などのリソースを予約・管理するシステムです。

- **Slackから予約できます**
- **Google Calendarからも予約できます**
- **予約の通知がSlackに届きます**

## 使い方

### 1. 初回登録（初めて使う時だけ）

研究室のDM以外の任意のSlackチャンネル（randomなど）で `/reserve` コマンドを実行すると、メールアドレスの入力画面が表示されます。

**普段Google Calendarを使っているメールアドレス**を入力してください。

- ✅ Google Calendarへのアクセス権が自動で付与されます
- ✅ Slackアカウントと紐づけられます
- ✅ 予約通知であなたがメンションされるようになります

### 2. 予約する（2つの方法）

#### 方法1: Slackから予約

```
/reserve
```

モーダルが開くので、予約したいリソース・期間・デバイスを選択して送信するだけです。

#### 方法2: Google Calendarから予約

Google Calendarで直接イベントを作成しても予約できます。

**GPUサーバーを予約する場合:**

イベントのタイトルに**使用するデバイス番号**を記入してください。

```
例:
0        → デバイス0だけ使う
0-2      → デバイス0,1,2を使う
0,3,5    → デバイス0,3,5を使う
0-1,6-7  → デバイス0,1,6,7を使う
```

メモなどを書きたい場合は、イベントの説明欄（description）に記入してください。

**部屋を予約する場合:**

デバイス番号の記入は不要です。普通にタイトルを書いてください。

```
例: 研究室ミーティング
```

### 3. 予約を変更・キャンセル

- **Slackから**: 通知メッセージの「🔄 更新」「❌ キャンセル」ボタンをクリック
- **Google Calendarから**: カレンダーのイベントを直接編集・削除

どちらの方法でも、変更があればSlackに通知が届きます。
